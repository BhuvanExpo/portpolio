# Anti-Gravity Portfolio

A modern, high-converting professional portfolio built with **React**, **Tailwind CSS v4**, and **Framer Motion**.

## Features

- **Anti-Gravity Aesthetic**: Smooth glassmorphism, floating elements, and interactive mesh gradients.
- **High-Converting Sections**:
  - **Hero**: Direct, human-centric value proposition.
  - **Services**: Specialized solutions for startup founders.
  - **Project Gallery**: Filterable case studies with interactive hover effects.
  - **About**: Personality-driven engineering philosophy.
  - **Contact**: Streamlined lead capture form.
- **Optimized Performance**: Built with Vite and Tailwind v4 for lightning-fast loads.
- **SEO Ready**: Full Open Graph and Meta tag support.

## Tech Stack

- **Framework**: React 19
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. **Clone the repo**:
   ```bash
   git clone https://github.com/BhuvanExpo/portpolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
